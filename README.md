# Knight Moves

Given the position of a knight on a chessboard, return the number of valid squares the knight can move to or the possible moves.  A knight moves in an 'L' shape : two squares in one direction (horizontal or vertical) , and one square in the perpendicular direction.  This means a knight can move up to 8 possible positions.

Taken from [freeCodeCamp Coding Challenge](https://www.freecodecamp.org/learn/daily-coding-challenge/2026-01-17)

## UX

**Getting Started**

Select the Column (e.g. `A`) and Row (e.g. `1`) from the dropdown menus.  You can also check the show moves, if you want to show the possible moves.  Click on the `Clear` button to empty the information.

**User Stories**

- As a user, I can select the column (e.g. `A`).
- As a user, I can select the row (e.g. `1`).
- As a user, I can check the shoe moves.
- As a user, if I click on the `Clear` button the information is empty.

## Features

Allows the user to select the column and row.  It allows the user to show the possible moves.

## Technologies

Uses HTML5, CSS3, JavaScript, Bootstrap 5.3.2 and jQuery 3.7.1.
